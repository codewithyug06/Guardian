Guardian integrates state-of-the-art Generative AI and Agentic workflows to deliver unique capabilities that outperform traditional monitoring tools:

* **Temporal Risk Projection**: Simulates risk trajectories over a 30-day window to visualize the "Future Risk Curve," enabling proactive threat neutralization.
* **Autonomous Self-Healing Loop**: Executes edge-level "Kill-Switches" to block non-compliant transactions while autonomously patching internal control libraries.

## üß† The Agent Swarm
Guardian's "Brain" is divided into specialized nodes that collaborate through a decentralized orchestration layer:

> **The Scout Node (Discovery)**
> Performs autonomous scanning and interprets the "spirit" of the law with verified confidence scoring. It replaces manual regulatory reading with instant semantic parsing.

> **The Sentry Node (Behavioral)**
> Monitors operational telemetry for behavioral anomalies and systemic, cross-domain violations. It identifies "Velocity Attacks" that standard rules miss.

> **The Architect Node (Strategy)**
> Maps findings to internal gaps and translates technical debt into quantified **Financial Liability ($)**. It connects the technical problem to the business bottom line.

> **The Prophet Node (Forecasting)**
> Simulates future risk trends and triggers the **Self-Healing Loop** for edge enforcement, visualizing how risks will compound over 30 days.

## ‚öôÔ∏è Installation & Setup

### Prerequisites
* Python 3.9+
* OpenAI API Key

### Quick Start

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/your-username/guardian.git](https://github.com/your-username/guardian.git)
    cd guardian
    ```

2.  **Environment Configuration**
    Create a `.env` file in the root directory and add your API credentials:
    ```env
    OPENAI_API_KEY=your_api_key_here
    ```

3.  **Install Dependencies**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Launch the Command Center**
    ```bash
    streamlit run app.py
    ```
